# Capstone_3
This project presents a comprehensive Power BI report developed for EmporiUm, a “virtual student bookstore” formed in 2022 through the merger of a college bookstore chain and an online student retail platform.
The goal of this report is to analyze sales performance over the last four years and provide actionable insights for regional leadership, including territory and store managers. My terriory is the Northeast region. With the manager being Michael Jarvis.\


The report is designed to:
* Analyze sales trends over time
* Evaluate sales performance across product categories
* Compare regional sales distribution by state
* Identify top-selling books and authors (excluding textbooks)
* Highlight key performance insights to support decision-making
